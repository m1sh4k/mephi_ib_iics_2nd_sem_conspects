

1. **Инкапсуляция** — объединение данных и методов для работы с ними в одном объекте. Из-за этого упрощен интерфейс взаимодействия. Внутренняя логика скрыта.
2. **Полиморфизм** — способность иметь одну и ту же функцию у разных классов с отличающейся реализацией. Разные реализации одного интерфейса.
3. **Абстракция** — выделение значимых характеристик объекта.
