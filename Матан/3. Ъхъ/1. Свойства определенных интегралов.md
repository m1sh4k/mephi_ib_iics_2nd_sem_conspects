## Свойство 2

Если $P'$ порождено добавлением в $P$ одной точки, то $\underline{\underline{S}}_{p'}\leq \underline{\underline{S}}_{p};\quad \overline{\overline{s}}_{p'}\geq \overline{\overline{s}}_{p}$
### Докво 
$$
\begin{align*}
&\underline{\underline{S}}_{p'}=\sum_{j=1;j\neq k}^{n_{p}}\sup f(x)\Delta x_{j}+\cancelto{ \leq  \sup\limits_{[x_{k-1};x_{k}]} f(x)}{ \sup_{[x_{k-1};x^{*}]} f(x) }\cdot\cancelto{ \ >\ 0 }{ (x^{*}-x_{k-1}) }+\cancelto{ 
 \leq  \sup\limits_{[x_{k-1};x_{k}]}f(x)}{ \sup_{[x^{*};x_{k}]}  f(x)}\cdot\cancelto{ \ >\ 0 }{ (x_{k}-x^{*}) }\leq \\
&\leq \sum_{j=1;j\neq k}^{n_{p}}\sup f(x)\Delta x_{j} +\sup_{[x_{k-1};k]} f(x)\cancelto{ \ x_{k}-x_{k-1} }{ (x^{*}-x_{k-1}+x_{k}-x^{*}) } = \underline{\underline{S}}_{p}
\end{align*}
$$
